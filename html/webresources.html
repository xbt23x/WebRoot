Web Resources

  1. SkillCrush
    This site explains the difference between front-end, back-end, and full stack development. This is a good resource to use because it tells us what goes into each of them with a decent amount of detail. Rating: 4/5

  2. FreeCodeCamp
    This is another website that can be used to show the differences between front-end, back-end, and full-stack development. This is a great resource because it gives a ton of detail and even offers courses and tutorials to follow. Rating: 5/5

  3. Medium article about web development
    This is an article by a developer who seems to have some decent experience working in the field. They explain more about what it is like to work in the industry and what kind of things you might be dealing with. Rating: 4.5/5

  4. w3Schools
    W3Schools is an amazing resource to use to learn more about web development or just coding in general. I've been using this website for several years to learn more from the tutorials offered from the site. Rating: 5/5

  5. GeeksforGeeks
    This site gives a brief explanation of what the differences are between the front end and the back end of development. This page will also provide tutorials on a variety of different topics that deal with coding. Rating: 4/5
